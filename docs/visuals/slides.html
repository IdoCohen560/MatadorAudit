AI-Generated Report Card

Claude AI transforms raw statistics into plain-English findings that any administrator can act on.

Finding: Hispanic/Latino students are 2.3x more likely to be denied financial aid renewal than white students with equivalent GPAs. This disparity persists after controlling for credit hours and enrollment status.

Root Cause: The "predicted persistence" variable in the allocation model correlates 0.78 with zip code, which correlates 0.71 with race/ethnicity in the CSUN student population. This constitutes proxy discrimination.

Recommended Action: Remove or re-weight the "predicted persistence" variable. Use the What-If Simulator to model alternative thresholds that maintain prediction accuracy while reducing demographic disparity.

Proxy Discrimination

AI systems don't need to use race directly to discriminate. MatadorAudit catches the hidden connections.

Correlation Matrix — Detected Proxies

Race Income
Zip Code 0.71 0.68
HS Ranking 0.54 0.61
Pred. Persistence 0.78 0.59
SAT Score 0.48 0.52

Red = strong proxy (>0.65) · Amber = moderate proxy (>0.45)

What this means: Even if the financial aid model never sees a student's race, it effectively "knows" race through zip code (r=0.71) and predicted persistence score (r=0.78).

Removing race from the input doesn't remove racial bias from the output. MatadorAudit makes this visible.

Ethics & Responsibility

🔒 Data Privacy

Operates on synthetic or de-identified data. No student PII sent to external APIs. Claude receives only statistical summaries, never raw records.

⚠️ Known Limitations

Fairness metrics can conflict mathematically. We present multiple metrics and let humans choose trade-offs. Small subgroup sizes are flagged.

🛡️ Human Oversight

No automated "debiasing" without review. The What-If Simulator is labeled as exploratory. All changes require institutional approval.

♿ Inclusive Design

WCAG 2.1 AA accessible dashboard. Plain-English reports. Multiple export formats for different stakeholders.

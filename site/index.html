The Problem

Universities deploy AI for admissions, financial aid, course recommendations, and plagiarism detection — but nobody checks if these systems treat all students fairly. Accuracy metrics can mask severe disparities across demographic groups.

The Gap

Existing fairness tools (Fairlearn, AIF360) require Python expertise. The people who need auditing most — department chairs, aid directors, admissions officers — cannot use them. MatadorAudit bridges this gap.

Technology

Microsoft Fairlearn + IBM AIF360 for metrics. Anthropic Claude API for plain-English reports. Microsoft Copilot & Google Gemini for follow-up Q&A. All open-source, under $500/year to operate.

Vision

Pilot at CSUN Financial Aid (Summer 2026), expand campus-wide, then scale to all 23 CSU campuses serving 460,000+ students. Open-source release for any university to deploy.
